# 社内文書検索スキル

ユーザーが社内文書に関する質問をした場合、このスキルを使って関連文書を検索し、その内容に基づいて回答してください。

## 使い方

ユーザーの質問から検索クエリを作成し、以下のJSON形式で呼び出してください:

```json
{
  "query": "検索したい内容の要点"
}
```

## パラメータ

- `query` (必須、string): ユーザーの質問をそのまま、または要点を抽出した検索クエリ。日本語のまま渡してください。

## 返却される内容

検索結果として、関連度の高い社内文書のスニペット（抜粋）が最大5件返されます。各結果には以下が含まれます:
- `index`: 文書番号
- `filename`: ファイル名
- `source`: ファイルのフルパス
- `snippet`: 文書の関連箇所（600文字以内）

## 回答ルール

1. 検索結果の内容のみを根拠にして回答してください。
2. 回答の最後に、根拠とした文書番号（例: 文書1, 文書3）を必ず記載してください。
3. 複数の文書に関連情報がある場合は、それらを統合して回答してください。
4. 検索結果から判断できない場合は「この質問に該当する情報は見つかりませんでした。」と回答してください。
5. 日本語で丁寧に回答してください。
6. 最後に、参照したファイル名の一覧を「📎 参考ファイル:」として列挙してください。

## 呼び出し例

ユーザーが「出張旅費の精算方法を教えて」と聞いた場合:

```json
{
  "query": "出張旅費 精算方法 手続き"
}
```
