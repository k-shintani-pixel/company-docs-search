# 社内文書検索スキル

## 使用ルール(必須)

**ユーザーからの質問に回答する前に、必ずこのスキルを呼び出してください。**

例外はありません。挨拶以外のすべての質問で、このスキルを最初に呼び出します。

## 呼び出し形式

以下のJSON形式で呼び出してください:

```json
{
  "query": "ユーザーの質問から抽出したキーワード"
}
```

### パラメータ

- `query` (必須、string): ユーザーの質問をそのまま日本語で渡してください。

## 検索結果の使い方

検索結果として、関連度の高い社内文書のスニペット(抜粋)が返されます。

1. 検索結果の内容**のみ**を根拠にして回答してください。
2. 回答の最後に、根拠とした文書番号(例: 文書1, 文書3)を必ず記載してください。
3. 複数の文書に関連情報がある場合は、それらを統合して回答してください。
4. 検索結果から判断できない場合は「この質問に該当する情報は見つかりませんでした。」と回答してください。
5. 日本語で丁寧に回答してください。

## 呼び出し例

### 例1
ユーザー: 「出張旅費の精算方法を教えて」
→ 呼び出し: `{"query": "出張旅費 精算方法"}`

### 例2
ユーザー: 「社内ネットワークの設定は?」
→ 呼び出し: `{"query": "社内ネットワーク 設定"}`

### 例3  
ユーザー: 「〇〇プロジェクトの進捗は?」
→ 呼び出し: `{"query": "〇〇プロジェクト 進捗"}`

どんな質問でも、まず検索してから回答してください。
